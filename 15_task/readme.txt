1) part1_s.c, part1_c.c. Две программы. Первая программа part1_s.c переопределяет сигнал(SIGUSR1). При получения сигнала выводит соответствующее сообщение. Для удобства отправки сигнала, программа выводит на экран свой PID. Вторая программа(part1_c.c) отправляет сигнал(SIGUSR1) программе по указанному PID. 

2) part2_s.c, part2_c.c. Две программы. Первая программа(part2_s.c) блокирует сигнал SIGINT(Ctrl + C). После блокировки сигнала программа уходит в бесконечный цикл, в котором можно попробовать вызвать сигнал SIGINT. Вторая программа(part2_c.c) аналогична программе part1_c.c, только она отправляет сигнал  SIGINT.

3) part3_s.c Программа реализует event-loop с помощью вызова sigwait(). Программа блокирует сигнал SIGUSR1 и переходит в бесконечный цикл ожидания сигнала, при получении сигнала на экран выводится сообщение о получении сигнала и осуществляется переход на следующую итерацию цикла. Проверить программу можно с помощью второй программы(part1_c.c) из первого задания.
